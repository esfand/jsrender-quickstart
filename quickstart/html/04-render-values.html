JsRender Quickstart - Render Values and Paths

Concepts

  • Render an object to a template: $("#myContainer").html($("#myTemplate").render(myObject));
  • Curly braces
  • Colon: {{:path}}
  • Using paths: {{:myObject.path}}
  • No encoding: {{:path}}
  • Html encoding: {{>path}}
  • Html encoding: {{html:path}}
